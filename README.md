# VanaVigyan Plant Research Website

A dynamic, responsive website for plant research and chemical compound analysis.

## Project Structure

## Features

### Home Page
- Responsive navigation with search functionality
- News ticker with latest research updates
- Hero section with call-to-action buttons
- Featured plants grid
- Research categories
- Latest research papers section
- Comprehensive footer

### Plant Pages
- Consistent header and footer with home page
- Detailed plant information including:
  - Chemical compounds
  - Medicinal properties
  - Uses and applications
  - Growing conditions
  - Research statistics
- Related plants sidebar
- Research links
- Responsive design for all devices

### Dynamic Features
- Real-time search with suggestions
- Interactive plant cards
- Smooth animations and transitions
- Mobile-friendly navigation
- SEO-optimized structure

## Installation

1. Download or clone the project files
2. Ensure all files maintain the directory structure
3. Open `index.html` in a web browser to view the home page
4. Navigate to individual plant pages through search or direct links

## Customization

### Adding New Plants
1. Add plant data to `plants.json`
2. Create new HTML file in `plants_data/` directory
3. Follow the structure of existing plant pages
4. Update related plants in sidebar as needed

### Styling
- Modify CSS variables in `:root` section of `style.css`
- Update colors, spacing, and typography as needed
- Add custom styles in appropriate CSS files

### Functionality
- Extend JavaScript in `script.js` for new features
- Add plant-specific functionality in `plant-pages.js`
